Python Programming Tutorial

Dive deep into Python — from the basics to real-world applications. This guide will take you from printing your first line to automating your first project.

Getting Started with Python

Python is an open-source language known for its simplicity and flexibility. Whether you're automating repetitive tasks, building web apps, or analyzing data, Python is a skill worth mastering. 

print("Hello, World!")

This simple command prints text to the screen — your first step into programming!

Core Concepts

Before building projects, understand the key components of the language:

  • Variables: Used to store values. Example: name = "Dean"
  • Data Types: Numbers, strings, lists, and dictionaries are core Python types.
  • Control Flow: Decide what runs with if, else, for, and while.
  • Functions: Define reusable logic blocks using def.
  • Modules: Organize large codebases and reuse utilities.

Practical Example: Automating Tasks

Let’s build a simple script that renames files in a folder:


import os

folder = "C:/Users/Dean/Documents/TestFolder"

for count, filename in enumerate(os.listdir(folder)):
    src = f"{folder}/{filename}"
    dst = f"{folder}/file_{count}.txt"
    os.rename(src, dst)

print("? All files renamed successfully!")

This example demonstrates automation — one of Python’s strongest suits. With just a few lines, you can handle hundreds of files in seconds.
