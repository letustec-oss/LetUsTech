📋 Tips for a good report

  • Copy the exact error message text if one appeared
  • Note the exact steps you took before it happened
  • Include the app version — check the title bar or Help → About
  • Mention if it happens every time or just occasionally
  • Say what file or data you were using (no need to share the file itself)
