How it works

From CSV to insights in seconds

1

Open your CSV

Drag & drop a CSV file onto the app or use File → Open (Ctrl+O). PapaParse reads it instantly — headers are auto-detected and every column is classified as numeric, categorical, or date automatically.

Works with any CSV — no format restrictions
2

Instant analysis

The moment your file loads, the app computes full descriptive statistics, scans for outliers using the 2.5σ rule, checks every column for missing values, and calculates Pearson correlations between all numeric column pairs.

3

Explore with charts

Switch to the Charts tab to visualise any combination of columns. Choose bar, line, scatter, or pie. See the distribution of numeric columns with histograms. Rank your categories by frequency.

4

Review data quality

The Data Quality tab gives you a full audit — missing percentages, unique counts, zero counts, and a status flag (OK / Some missing / High missing / Constant) for every column.

5

Ask AI questions (optional)

Add a free Anthropic API key in Settings and the AI Insights tab unlocks. Ask Claude "What story does this data tell?" or "Which columns should I use for forecasting?" — your raw data never leaves your machine, only column-level statistics are sent.

Only column statistics sent — not your raw data
6

Export your report

Click Export Report (Ctrl+E) to generate a polished standalone HTML file with all your statistics, insights, and quality audit. Open it in any browser, email it, or drop it in a presentation.

What's included

Free features vs AI-powered

Everything core is completely free. AI Insights requires an Anthropic API key — typical usage costs pennies per session.
