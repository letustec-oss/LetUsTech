Remote Support

How does remote support work?

+
We use Windows Quick Assist (built into Windows 10 and 11) to connect to your machine securely. You run Quick Assist, select "Get assistance", and share the 6-digit code with us. We can then see and control your screen — with your permission — to diagnose and fix the issue. You can end the session at any time.

Is remote access safe?

+
Yes. We use Microsoft's Quick Assist, which is encrypted and built into Windows. You generate a one-time code to share with us — we cannot access your machine without it. You can see everything we do on screen in real time and stop the session instantly if needed.

What can you fix remotely?

+
Most software issues can be resolved remotely — this includes slow PCs, virus removal, software installation, printer problems, email setup, Microsoft 365 issues, Wi-Fi troubleshooting, and general Windows problems. Hardware faults that require physical access will need an on-site visit.

How quickly can you respond?

+
We aim to respond to all enquiries within one hour during business hours (Monday–Friday 9am–5:30pm). Managed clients on a support plan receive priority response. Emergency cover is available for managed clients outside of normal hours.
