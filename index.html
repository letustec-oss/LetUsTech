FAQ

Common questions

Everything you need to know about LetUsTech.

General

Are all programs really free?

+
Yes — 100% free, forever. No trials, no premium tiers, no hidden costs. We run on donations and ads.

Do I need to create an account?

+
No. Download what you need and use it immediately. No sign-up, no personal info required.

How do you sustain LetUsTech?

+
Voluntary donations from users who love what we do, plus display ads. Every bit helps cover hosting and dev time.
Technical

What is Python and do I need it?

+
Python is a popular, free programming language. Most downloadable tools need Python 3.8+ installed. Get it free at python.org. Browser games need nothing extra.

What operating systems are supported?

+
Windows, macOS, and Linux for desktop tools. Browser games work on any device with a modern browser — phone, tablet, or desktop.

Are your programs safe to download?

+
Yes. Clean code, no malware or tracking. We still recommend scanning all downloads with antivirus software as a best practice.

Can I use the programs offline?

+
Yes — once downloaded, most tools work completely offline. A few features (like live data fetching) need internet, but the core tools do not.
Games

Do I need to download or install games?

+
No — all games run directly in your browser. Click "Play Now" and you're in instantly. No installs, no plugins, no accounts.

Will you add more games?

+
Yes! Code Breaker is coming soon, and we're always working on new titles. Got an idea? Send it our way.
Privacy & Security

What data do you collect?

+
Minimal: basic analytics via Google Analytics, and cookies for ads via Google AdSense. We do not collect personal information unless you contact us directly. See our Privacy Policy.

Do you sell my data?

+
Never. We don't sell, rent, or trade your data to anyone, ever.
Support & Donations

How do I get support?

+
Email letustec@gmail.com — we typically respond within 24 hours. Include which program or game you're using, your OS, and a description of the issue.

How can I support LetUsTech?

+
Donate via PayPal or UK bank transfer, share us with friends, report bugs, or send feedback. Every bit helps keep these tools free.

Will you add more programs?

+
Absolutely — constantly building based on user needs. Upcoming: PDF tools, file converters, and advanced data visualisation.
