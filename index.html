FAQ

Common questions

Everything you need to know about LetUsTech.

General Questions

Are all programs really free?

+

Yes — 100% free, forever. No trials, no premium tiers, no hidden costs. We run on donations and ads.

Do I need to create an account?

+

No. Download what you need and use it immediately. No sign-up, no personal info required.

How do you sustain LetUsTech?

+

Voluntary donations from users who love what we do, plus display ads. Every bit helps cover hosting and dev time.

Technical Questions

What is Python?

+

A popular, free programming language. Most of our downloadable tools need Python 3.8+ installed. Get it free at python.org.

What operating systems are supported?

+

Windows, macOS, and Linux. Browser games work on any device with a modern browser.

Are your programs safe?

+

Yes. Clean code, no malware or tracking. We still recommend scanning all downloads with antivirus as best practice.

Games & Support

Do I need to download games?

+

No — all games are browser-based. Click "Play Now" and you're in instantly. No installs, no plugins.

How do I get support?

+

Email [email protected] — typically responding within 24 hours. Include which program you're using, your OS, and details of the issue.

Do you update programs regularly?

+

Yes — monthly updates or as needed based on user feedback. Download the latest version from our site anytime.

Privacy & Security

What data do you collect?

+

Minimal: basic analytics, IP address, browser info. We do NOT collect personal info unless you contact us. See our Privacy Policy.

Do you sell my data?

+

Never. We don't sell, rent, or trade your data to anyone.

Can I use programs offline?

+

Yes — once downloaded, most tools work completely offline unless they specifically need to fetch web data.

Program Features

What does Data Analyzer do?

+

Analyzes CSV files with statistical insights — averages, medians, standard deviations — plus charts and graphs.

Is web scraping legal?

+

Legal when done ethically. Always check a site's robots.txt and terms of service, respect rate limits, and only scrape public data.

Can I modify the programs?

+

Yes, for personal use. Python is human-readable and easy to customise. Please don't redistribute modified versions commercially.

Support & Donations

How can I support LetUsTech?

+

Donate via PayPal or bank transfer, tell friends, report bugs, or leave feedback. Everything helps!

Are donations tax-deductible?

+

No — we're not a registered charity. We're a small team making free software, supported by voluntary contributions.

Will you add more programs?

+

Absolutely — constantly building based on user needs. Upcoming: file converters, social media automation, advanced data visualisation.
