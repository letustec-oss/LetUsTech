Connect to Firebase

LetUsTech RMM uses Firebase as a free relay between your client devices and this dashboard. Set it up once and you're done.

  1. Go to console.firebase.google.com and sign in with Google
  2. Click Add project → name it "letutech-rmm" → Create
  3. Go to Build → Realtime Database → Create database → Start in test mode
  4. Go to Project Settings (gear icon) → General
  5. Scroll to Your apps → Web app (</>) → Register app → copy the config below
