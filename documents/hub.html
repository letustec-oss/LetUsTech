New Client Onboarding — Business Support Plan

Follow these steps in order when someone buys the £299/month plan

1
Send the service agreement
Open documents/service-agreement.html → fill in client details → Print to PDF → email it to them
2
Client signs the agreement
They sign and return a copy — keep it on file before starting any work
3
Send the GoCardless payment link
https://pay.gocardless.com/BRT01KWCB9XKN299KPQZN5H8ZRNZQ — client sets up Direct Debit (takes 2 mins)
4
Add client to Client Manager
Open the Client Manager app → Add Client → fill in their name, company, email, phone, devices
5
Book onboarding remote session
Schedule 30–60 min call — you'll use Quick Assist or Atera to connect to their machine
6
Install Atera agent on their machines
Log into Atera → Devices → Add Device → run the installer on each client machine. This enables proactive monitoring.
7
Get Microsoft 365 delegated admin access
Client goes to admin.microsoft.com → Settings → Partner relationships → Add your Microsoft account as admin
8
Run initial security check
Open Support Tool → PC Diagnostics + Virus & Malware tabs → check Windows Update, Defender status, disk health
9
Document everything in Client Manager
Add job: "Onboarding" → log device list, admin passwords (in Bitwarden), network details, M365 tenant name
10
Confirm setup with client
Email them confirming they're live — include your support email, response hours, and how to raise a ticket
